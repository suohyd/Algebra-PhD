    Proposition 1.3.3.

    Keep notations of DefinitionÂ 1.3.1 above. Let for \(g\in G\text{,}\) \(\chi_g\) be the characteristic function of \(g\text{,}\) i.e., \(\chi_g(h)=0\) if \(h\neq g\) and \(\chi_g(g)=1\text{.}\) Then, set \(\{\chi_g:g\in G\}\) is an \(R\)-basis of \(RG\text{.}\) In particular, \(RG\) is a free \(R\)-module.
    ðŸ”—
    Furthermore, \(G\to RG\) given by \(g\mapsto\chi_g\) is a monoid homomorphism. In particular, \(\chi_g\chi_h=\chi_{gh}\) and if \(e\in G\) is the identity then, \(\chi_e\) is the unity in \(RG\text{.}\)

    Proof.

    Suppose that \(\sum\chi_{g_i}r_i=0\) for some \(g_i\in G\) and \(r_i\in R\text{.}\) Evaluating at \(g_j\text{,}\) we get \(r_j=0\text{.}\) Thus, the set is linearly independent. Now, let \(\phi\in RG\text{.}\) Then, we can write \(\phi=\sum\chi_g\phi(g)\) where the sum is over all \(g\in G\) such that \(\phi(g)\neq 0\text{.}\) This shows that the set spans \(RG\text{.}\) Thus, it is an \(R\)-basis of \(RG\text{.}\)
    ðŸ”—
    Direct calculation shows that \(\chi_g\chi_h=\chi_{gh}\) for all \(g,h\in G\text{.}\)
    ðŸ”—
    We show the associativity of multiplication. Let \(\phi,\psi,\zeta\in RG\text{.}\) Then, there exists \(r_i,s_j,t_k\in R\) such that \(\phi=\sum\chi_{g_i}r_i\text{,}\) \(\psi=\sum\chi_{g_j}s_j\text{,}\) \(\zeta=\sum\chi_{g_k}t_k\text{.}\) Therefore, we obtain the following.
    \begin{align*} \left(\phi\psi\right)\zeta\amp =\left(\left(\sum\chi_{g_i}r_i\right)\left(\sum\chi_{g_j}s_j\right)\right)\sum\chi_{g_k}t_k\\ \amp = \sum\left((\chi_{g_i}\chi_{g_j})\chi_{g_k}\right)(r_is_jt_k)\\ \amp =\sum\chi_{g_i}\left(\chi_{g_j}\chi_{g_k}\right)(r_is_jt_k) \\ \amp =\phi\left(\psi\zeta\right) \end{align*}
    This shows the associativity of multiplication.

    For any \(\chi_{g_i}a_i\in RG\) we get the following.
    \begin{align*} \left((\chi_{g_i}a_i)\chi_e\right)(h) \amp =\sum\chi_{g_i}a_i(x)\chi_e(x^{-1}h) \end{align*}
    The above equation is nonzero only when \(x=g_i\) and \(x^{-1}h=e\text{.}\) This implies that \(h=g_i\text{.}\) Thus, we get \(\left((\chi_{g_i}a_i)\chi_e\right)(h)=\chi_{g_i}a_i(h)\text{.}\) Similarly, we can show that \(\left(\chi_e(\chi_{g_i}a_i)\right)(h)=\chi_{g_i}a_i(h)\text{.}\) This shows that \(\chi_e\) is the unity in \(RG\text{.}\)
